# **App Name**: BudgetWise

## Core Features:

- Expense Tracking: Input and categorize expenses manually or through connecting bank accounts.
- Category Management: Create and manage spending categories to classify expenses.
- Budget Goals: Set monthly budget goals for different spending categories.
- Visual Reporting: Visualize spending habits through interactive charts and graphs. The tool generates the charts and selects which information to include, to give a well rounded picture to the user of their finances.
- Dashboard: Display key financial insights and summaries on a clean, customizable dashboard.

## Style Guidelines:

- Primary color: A calming, muted blue (#6699CC), reminiscent of trustworthiness and financial stability.
- Background color: Light gray (#F0F0F0), to provide a clean and unobtrusive backdrop for the data.
- Accent color: A contrasting, vibrant orange (#FFA07A) to highlight important actions and notifications.
- Clean, sans-serif fonts (e.g., Open Sans, Lato) for easy readability of financial data.
- Simple, line-based icons to represent spending categories and actions.
- Balanced and spacious layout with clear sections for expenses, categories, budgets, and insights.
- Subtle transitions and animations when updating or displaying new financial data.
